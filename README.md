# せどり新着アラート

Web版・自分専用の新着通知ツールです。

キーワードを登録しておくと、対応ショップで見つかった商品を記録し、条件に合うものをDiscordへ通知します。

## 対応ショップ

- セカンドストリートオンライン
- オフモール
- まんだらけ
- 駿河屋

## 主な機能

- キーワード登録
- 価格下限・価格上限
- 表記ゆれ / 追加一致語
- 除外語
- 店舗別の状態ランク条件
- 商品一覧
- 通知履歴
- Discord Webhook通知
- 実ショップ取得テスト

## 起動方法

Windowsで以下をダブルクリックします。

```text
00_START.bat
```

起動後、ブラウザで以下を開きます。

```text
http://127.0.0.1:8000/
```

初期ログイン:

```text
admin@example.com
password123
```

## 注意

このツールはローカルPC上で動く自分専用ツールです。
PCの電源が切れている間は、巡回やDiscord通知は動きません。
